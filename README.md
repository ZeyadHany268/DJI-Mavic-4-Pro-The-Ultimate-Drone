# 🚁 DJI Mavic 4 Pro Landing Page



A modern responsive product landing page for the **DJI Mavic 4 Pro** drone.  
The project presents a professional product showcase with specifications comparison, customer reviews, related products, and a responsive navigation experience.

---

## 📌 Project Description

This project is a front-end product page designed to showcase a premium drone product.

It includes:

- Product hero section
- Product pricing and actions
- Detailed specifications comparison table
- Customer reviews section
- Related accessories/products section
- Responsive footer
- Mobile-friendly navigation menu

The design focuses on a clean UI, responsive layout, and modern user experience.

---

## ✨ Features

- ✅ Responsive design for desktop, tablet, and mobile
- ✅ Fixed navigation bar
- ✅ Mobile hamburger menu with JavaScript
- ✅ Smooth scrolling between sections
- ✅ Product showcase section
- ✅ Discount badge and pricing display
- ✅ Comparison table with product highlights
- ✅ Customer ratings and review cards
- ✅ Product accessory cards
- ✅ Hover animations and interactive effects
- ✅ Responsive table scrolling on small screens
- ✅ Modern card-based UI layout

---

- HTML5
